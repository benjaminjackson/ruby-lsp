# Ruby LSP Plugin for Claude Code

Dead-simple Ruby LSP integration for Claude Code.

## What This Does

Connects Claude Code to the Ruby LSP server, providing real-time code intelligence:
- Syntax error diagnostics
- Go-to-definition
- Hover documentation
- Code navigation

## Prerequisites

```bash
gem install ruby-lsp
```

## Installation

```bash
/plugin marketplace add benjaminjackson/ruby-lsp
/plugin install ruby-lsp@ruby-lsp-marketplace
```
